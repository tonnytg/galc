# galc
Google Cloud ACL Backup
DOC: https://cloud.google.com/storage/docs/access-control/lists



Permission:
- READER
- WRITER
- OWNER

Scope:
- Email
- Groups (work-group@googlegroups.com)
- allUsers -> Fora da organização
- allAuthenticatedUsers - Fora da Organização


Maximum:
- 100 ACL each object
